## Summary

С помощью методов Заказы DBW (Доставка курьером WB) вы можете: - получать информацию о [сборочных заданиях](/openapi/orders-dbw#tag/Sborochnye-zadaniya-DBW), управлять статусами и отменять сборочные задания - получать, добавлять, редактировать и удалять [метаданные](/openapi/orders-dbw#tag/Metadannye-DBW) сборочных заданий Узнать, как использовать методы в бизнес-кейсах, можно в инструкции по работе с заказами DBW
